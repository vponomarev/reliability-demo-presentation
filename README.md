# Демо-презентация на основе [reveal.js](https://revealjs.com/)

## Зависимости
- Для запуска презентации необходимо установить `npm` (nodeJS)


## Быстрый старт
- `npm install`
- `npm start`
- открыть в браузере http://localhost:8000/seminars/reliability/
- нажми клавишу `s` для запуска презентации
- https://revealjs.com/installation/
